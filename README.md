# Contas Viagem Moto 2026 — mini-site

Este mini-site (index.html + app.js + styles.css) é standalone e funciona offline.

## Funcionalidades
- Alterar nº de pessoas, consumo, preço do litro e refeições/dia.
- Editar valores por dia (noite, garagem, kms) e ver totais a atualizar.
- Layout moderno + mobile-friendly + modo claro/escuro.

## Como usar
Abrir `index.html` no browser.

## Publicar online
Qualquer hosting de ficheiros estáticos (GitHub Pages, etc.).
